## Customization

### Change Colors
Find and replace `#ff6b00` with your preferred color throughout the CSS:
```css
/* Primary color */
color: #ff6b00;
```

### Update Business Info
Edit the contact section in `index.html`:
```html
<p><strong>Address:</strong> Your Location Here</p>
<p><strong>Phone:</strong> +1 (555) 123-4567</p>
<p><strong>Email:</strong> info@barlacanalet.com</p>
```

### Change Hero Background
Modify the background image URL in `styles.css`:
```css
background: linear-gradient(rgba(0,0,0,0.4), rgba(0,0,0,0.4)), url('YOUR_IMAGE_URL') center/cover;
```

### Update Menu Items
Edit the cards in the menu section with your offerings.
